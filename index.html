How does javascript works?

1. জাভাস্ক্রিপ্ট এর কোড যখন ব্রাউজারের কাছে যায়, ব্রাউজার সেটাকে জাভাস্ক্রিপ্ট এর v8 engine কাছে পাঠায়, v8 engine কোডকে compile করে machine language পরিণত করে, মেশিন সেটা বুঝতে পারে এবং আমাদের display তে output দেয় ।

2. জাভাস্ক্রিপ্ট কোড Just-In-Time (JIT) compiler এর মাধ্যমে v8 engine এ compiled হয়।

3. জাভাস্ক্রিপ্ট synchronous and asynchronous উভয় way তেই কাজ করে।

4. জাভাস্ক্রিপ্ট শুরুতেই global execution context তৈরী করে। যার মধ্যে window,this,variable objects ,scope-chain নামে বিভিন্ন objects থাকে।

5. জাভাস্ক্রিপ্ট Single-threaded way তে code execute করে।
